# Setup

## Cutting Edge 
1. `mkdir /opt/euphoria ; cd /opt/euphoria`
2. `wget https://github.com/thoughtpivot/euphoria/archive/master.tar.gz`
3. `tar -xvf euphoria-master.tar.gz`
4. `./Makefile.sh`

Start Euphoria
  sudo ./euphoria.sh

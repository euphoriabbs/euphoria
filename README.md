# euphoriabbs

A Modern Bulletin Board System Built on Synchronet.

# Download & Install
	./Build

# Configure
	./euphoria-scfg
and...
	vim euphoria-sbbs.ini
# Server Start
	./euphoria-sbbs

Linux

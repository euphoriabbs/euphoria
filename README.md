# Euphoria

A Modern Bulletin Board System built on Synchronet BBS.

# Setup

## Cutting Edge 
1. `sudo mkdir /opt/euphoria ; cd /opt/euphoria`
2. `sudo wget https://github.com/thoughtpivot/euphoria/archive/master.tar.gz`
3. `sudo tar -xvf master.tar.gz`
4. `sudo ./Makefile.sh`

Start Euphoria
  sudo ./daemon.sh
